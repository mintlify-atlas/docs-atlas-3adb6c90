# React QR Code

A `<QRCode />` component for React and React Native. This library works with React and React Native (using [React Native SVG](https://github.com/react-native-svg/react-native-svg)).

## Key Features

<CardGroup cols={2}>
  <Card title="Simple Component" icon="react">
    Drop-in `<QRCode />` component that works with minimal configuration
  </Card>
  <Card title="UTF-8 Support" icon="language">
    Native support for non-ASCII text including Korean, Japanese, Chinese, and emoji
  </Card>
  <Card title="Highly Customizable" icon="palette">
    Customize colors, size, error correction level, and more
  </Card>
  <Card title="SVG-Based" icon="vector-square">
    Renders as scalable SVG for crisp quality at any size
  </Card>
  <Card title="Cross-Platform" icon="mobile">
    Works seamlessly on web, iOS, and Android
  </Card>
  <Card title="Spec Compliant" icon="check">
    Adheres to the official QR spec - stores up to 2953 characters
  </Card>
</CardGroup>
